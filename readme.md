### CodeMore QRCode 基于 colorful-qrcode的分支
----

1. 颜色是随机生成的深彩色
1. 使用Canvas生成，不联网也可以使用哦
2. 点开Popup，点击生成的二维码或者`回车`，然后可以手动输入需要的字符，再次`回车`生成
3. 输入为空时，默认生成当前页面的二维码
4. 还是喜欢黑色？设置自己找找吧
